### 🎯 Visão do Projeto

**Este repositório está se transformando no sistema principal!** 

Começamos aprendendo Git e agora estamos evoluindo para um sistema web completo usando Python e Flask. Cada tarefa adiciona funcionalidades e conhecimento ao projeto, criando um sistema real que todos poderão usar e contribuir.

#### Tecnologias que usaremos:
- **Git/GitHub**: Controle de versão e colaboração
- **Python 3.13+**: Linguagem principal  
- **Flask**: Framework web
- **Poetry**: Gerenciamento de dependências
- **SQLAlchemy**: Banco de dados
- **Bootstrap**: Interface responsiva
- **Pytest**: Testes automatizados
